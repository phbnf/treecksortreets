How to Play

The top root hash of the tree is mismatching (red) because one of the leaves at the bottom has been modified.

Scan: Click any node in the tree to audit it. The node will turn:
🔴 Red if the tampered leaf is a descendant of this node.
🟢 Green if all descendants of this node are authentic.

Goal: Find the modified leaf using no more than 4 scans, select it, and click "Declare Imposter".
